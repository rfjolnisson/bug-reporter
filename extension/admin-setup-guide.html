How to Use

For End Users:

  1. Visit any page (works in Salesforce!)
  2. Click the bug icon üêõ in Chrome toolbar
  3. Wait 1-2 seconds for screenshot capture
  4. Fill in the issue title and description
  5. Click "Submit to JIRA"
  6. Get a link to the created JIRA issue!
üí° What Gets Captured The extension automatically captures ALL console logs (errors, warnings, info) and takes a screenshot of the page when you click the icon.

Quick Troubleshooting

Extension not appearing? Make sure you selected the extension folder (contains manifest.json). Reload the page and try again.
Can't submit to JIRA? The backend server must be running. Contact the Kaptio team to verify the server is accessible at port 3519.
No console logs showing? Refresh the page after installing the extension. It only captures logs from when the page loads.
